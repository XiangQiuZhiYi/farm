## 1. 土壤与田地系统

### 1.1 首发土壤分组

本轮文档首批只展开以下 3 组：

| 分组 | 现实区域参考 | 玩法定位 |
|------|--------------|----------|
| 水稻土区 | 长江中下游、四川盆地、南方各省 | 初始主区域，强调水田、水生作物、湿润环境 |
| 褐土 / 潮土组 | 华北平原、鲁豫冀晋陕关中 | 过渡区域，强调旱地、蔬菜、豆类与轮作 |
| 黑土区 | 黑龙江、吉林、辽宁北部 | 高肥力区域，强调高产粮食与冷凉作物 |

### 1.2 核心田地类型

首发重点田地类型如下：

1. **水田**：适合高湿、高水位植物
2. **旱地**：适合常规蔬菜、豆类、部分经济作物
3. **梯田**：适合坡地农业与特殊区域管理

### 1.3 环境变量

每块田地至少需要记录以下环境参数：

| 参数 | 说明 |
|------|------|
| 土壤类型 | 决定根本适配关系 |
| 含水状态 | 决定水生 / 旱生植物的正常成长 |
| 温度区间 | 决定是否满足作物活跃生长条件 |
| 季节月份 | 决定是否处于适宜播种与成熟阶段 |
| 地势类型 | 平地、低洼地、梯田等 |
| 改良等级 | 决定土壤肥力、稳定性、保水能力 |

### 1.4 土地适配规则

植物与土壤的关系分为 3 类：

1. **最适土壤**：成长最快、品质最好、最容易稳定成熟
2. **兼容土壤**：可以种，但速度、产量或品质会下降
3. **非适配土壤**：只能发芽或勉强存活，无法正常成熟

### 1.5 强绑定植物规则

每个土壤分组至少需要 1~2 种植物满足以下条件：

- 只能在该土壤中正常成长
- 换到其他土壤时可以发育异常、停滞或无法成熟
- 用于强化“适地种植”的策略差异

## 2. 区域土地摘要

| 区域 | 地块特征 | 经营定位 | 代表性田地 |
|------|----------|----------|------------|
| 水稻土区 | 高湿、适合淹水与湿田管理，兼顾旱化改造 | 初始教学区域，强调水分控制与轮作启蒙 | 淹水田、浅水湿田、旱化排水田 |
| 褐土 / 潮土组 | 旱地与灌区并存，强调排水与墒情差异 | 中后期过渡区域，强调两熟制与旱涝分化 | 旱地、灌区田、排水良好经济作物田 |
| 黑土区 | 高肥力、土层深厚、偏冷凉气候 | 中后期高价值区域，强调高产大田和冷凉窗口 | 高肥大田、灌溉田、疏松整地田 |

### 2.1 区域地块规模

| 区域 | 初始土块 | 初始组成 | 扩张上限 |
|------|----------|----------|----------|
| 水稻土区 | 6 格 | 3 淹水田 + 2 浅水湿田 + 1 旱化排水田 | 18 格 |
| 褐土 / 潮土组 | 5 格 | 3 旱地 + 1 灌区田 + 1 排水良好田 | 15 格 |
| 黑土区 | 4 格 | 2 高肥大田 + 1 灌溉田 + 1 疏松整地田 | 12 格 |

设计意图如下：

- **前期区域格子更多**，方便教学、容错和多品种试种。
- **后期区域单格价值更高**，但初始格子更少，迫使玩家谨慎经营。
- **区域差异不只体现在植物表里**，也体现在“可立即使用的地块规模”和“扩张压力”上。

## 3. 分区域土地经营重点

### 3.1 水稻土区

水稻土区以**长江中下游、四川盆地、南方各省**为主要现实参考，适合构建玩家的初始区域。

土地经营重点：

- **淹水田**：服务水稻、莲藕、茭白、荸荠、菱角、慈姑、芡实等高水位作物
- **浅水 / 湿田**：适合空心菜、水芹等短中周期补位植物
- **排水田（旱化水稻土）**：支撑油菜、小麦、蚕豆、马铃薯、大豆、芋头等轮作
- 可直接做教学轮作：**稻-油、稻-麦、稻-豆、稻-藕**

### 3.2 褐土 / 潮土组

褐土 / 潮土组以**华北平原、鲁豫冀晋陕关中**为主要现实参考，强调旱地粮食、经济作物和两熟制结构。

土地经营重点：

- **两熟制结构**：小麦-玉米、小麦-花生可作为核心轮作
- **旱地与灌区差异**：褐土偏旱地经营，潮土更适合灌区作物
- **高价值作物承载地**：棉花、芝麻、西瓜适合作为区域解锁奖励
- **策略重点**：排水、墒情控制、冬小麦与夏作衔接

### 3.3 黑土区

黑土区以**黑龙江、吉林、辽宁北部**为主要现实参考，强调高肥力、冷凉气候和高产粮经作物。

土地经营重点：

- **黑土高肥力**：玉米、大豆、马铃薯等高产作物收益更突出
- **冷凉优势**：亚麻、甘蓝、胡萝卜等作物更容易保持稳定品质
- **高价值专属**：甜菜、亚麻可承担黑土区扩张奖励
- **策略重点**：肥力利用、冷凉窗口与大田高产

## 4. 土地改良与扩张主线

长期成长主线围绕土地经营展开：

1. 改良初始土地
2. 提升亩产与品质
3. 建立稳定售卖循环
4. 完成图鉴与品种积累
5. 满足更高阶区域的解锁条件
6. 向更难经营、但更高价值的土地扩张

当前确定的扩张门槛为：

1. **植物解锁**：初始只解锁水稻，其他植物按“累计赚到过的资金”达到门槛后永久解锁。
2. **区域解锁价格**：玩家必须拥有足够当前资金，才能支付区域解锁费用。
3. **作物图鉴 / 品种完成度**：玩家需要完成上一区域 **10 种植物图鉴**，才能进入下一区域。

当前确定的区域解锁顺序与价格如下：

| 区域 | 解锁顺序 | 区域解锁价格 | 图鉴前置 | 区域内新增地块价格 |
|------|----------|--------------|----------|--------------------|
| 水稻土区 | 1 | 0 | 无 | 90 / 格 |
| 褐土 / 潮土组 | 2 | 1600 | 水稻土区图鉴 ≥ 10 | 180 / 格 |
| 黑土区 | 3 | 4200 | 褐土 / 潮土组图鉴 ≥ 10 | 320 / 格 |

### 4.1 购买与出售流程

#### 区域与地块购买

1. **区域购买**必须同时满足：
   - 当前持有资金足够支付区域解锁价格
   - 已完成上一区域 **10 种植物图鉴**
2. 区域解锁后，玩家可以继续支付“区域内新增地块价格”来扩张地块数量。
3. 区域购买解决的是“进入哪个土区”，地块购买解决的是“在该土区能经营多少格土地”。

#### 作物出售

1. 只有**已经成熟并收获入库**的作物可以出售。
2. 首版采用**固定售价**，不引入市场波动。
3. 实际售卖收入 = `作物售价/份 × 实际产量`。
4. 实际产量会受到土壤适配、季节时机、收获时机和浇水管理共同影响。

### 4.2 种植与浇水流程

#### 种植

1. 玩家先选择一块空地。
2. 种植面板只展示当前已解锁植物，并标注对应土地适配与季节信息。
3. 确认种植后扣除 1 份种子或种苗，地块进入成长阶段。
4. 不适配土地仍允许种植，但会明确提示减产风险或成熟失败风险。

#### 浇水

1. 每块地统一维护 `缺水 / 正常 / 过湿` 三档含水状态。
2. **旱地类**以主动浇水为主：玩家浇水后提升含水等级。
3. **水田类**以维持水层为主：目标是让地块保持在适宜的浅水状态，而不是无限加水。
4. 水分会随时间变化，玩家需要根据作物和土地类型决定补水或排水。
5. 含水状态直接影响成长速度、最终产量和失败概率，是土地经营的重要策略点。

区域价值梯度如下：

- 初始区域更稳定，容错更高
- 新区域更难管理，对土壤和季节适配要求更高
- 越晚解锁的区域，代表性作物价值越高，种植策略也越复杂
