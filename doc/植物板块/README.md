## 1. 植物资料整理规则

### 1.1 每种植物需整理的字段

| 字段 | 说明 |
|------|------|
| 植物名称 | 中文常用名 |
| 植物类别 | 粮食 / 蔬菜 / 水生作物 / 经济作物 / 茶果 / 药材 / 香草等 |
| 最适土壤 | 最适合正常成长的土壤类型 |
| 兼容土壤 | 可种植但表现不如最适土壤的类型 |
| 水分条件 | 水田 / 潮湿 / 中湿 / 偏干 |
| 适温范围 | 适合的主要生长温度 |
| 现实生长期 | 现实农业常见生长周期 |
| 游戏成长时间 | 按规则换算后的游戏分钟 |
| 季节月份 | 适宜播种与生长月份 |
| 玩法备注 | 是否适合作为强绑定植物、是否适合早期教学等 |

### 1.2 再生与多年生规则

#### 再生作物

- 首次成熟按完整周期换算
- 后续再生按“再次采收间隔”单独换算
- 再生周期建议不低于 **240 分钟**

#### 多年生作物

- 定植 / 成园期按完整现实周期换算
- 达到成龄后，进入定期产出模式
- 非产季只保留植株，不产出

## 2. 首批资料整理目标

> 本章节用于承接资料收集结果，并整理成正式文档表格，不直接堆砌原始来源。

### 2.1 解锁与价格规则

1. **初始只解锁水稻**。
2. **植物解锁**：按“累计赚到过的资金”达到门槛后永久解锁，不消耗解锁费用。
3. **图鉴解锁**：默认按“成功收获过一次该植物”计入图鉴。
4. **区域顺序**：`水稻土区 → 褐土 / 潮土组 → 黑土区`。
5. **区域解锁**：必须同时满足“当前持有资金足够支付区域解锁价格”以及“完成上一区域 10 种植物图鉴”。

### 2.2 区域土地价格

| 区域 | 解锁顺序 | 区域解锁价格 | 图鉴前置 | 区域内新增地块价格 |
|------|----------|--------------|----------|--------------------|
| 水稻土区 | 1 | 0 | 无 | 90 / 格 |
| 褐土 / 潮土组 | 2 | 1600 | 水稻土区图鉴 ≥ 10 | 180 / 格 |
| 黑土区 | 3 | 4200 | 褐土 / 潮土组图鉴 ≥ 10 | 320 / 格 |

## 3. 水稻土区植物

水稻土区以**长江中下游、四川盆地、南方各省**为主要现实参考，适合构建玩家的初始区域。

### 3.1 强绑定植物

- **水稻**：只能在淹水型水稻土中正常生长
- **莲藕**：只能在肥泥型水稻土 / 藕田环境中正常生长

可作为接近专属的候选：

- 茭白
- 荸荠

### 3.3 玩法提炼

- **淹水田**：水稻、莲藕、茭白、荸荠、菱角、慈姑、芡实表现最佳
- **浅水 / 湿田**：空心菜、水芹适合做短中周期补位
- **排水田（旱化水稻土）**：油菜、小麦、蚕豆、马铃薯、大豆、芋头可形成轮作
- 可直接做教学轮作：**稻-油、稻-麦、稻-豆、稻-藕**

- **黑土高肥力**：玉米、大豆、马铃薯等高产作物收益更突出
- **冷凉优势**：亚麻、甘蓝、胡萝卜等作物更容易保持稳定品质
- **高价值专属**：甜菜、亚麻可承担黑土区扩张奖励
- **黑土区策略重点**：肥力利用、冷凉窗口与大田高产

### 5.3 玩法提炼

- **两熟制结构**：小麦-玉米、小麦-花生可作为核心轮作
- **旱地与灌区差异**：褐土偏旱地经营，潮土更适合灌区作物
- **高价值作物**：棉花、芝麻、西瓜适合作为区域解锁奖励
- **褐土 / 潮土组策略重点**：排水、墒情控制、冬小麦与夏作衔接
